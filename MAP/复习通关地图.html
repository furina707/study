🎯 核心公式速查表

📖 泰勒/麦克劳林展开
eˣ = 1 + x + x²/2! + x³/3! + ··· + xⁿ/n!
sin x = x - x³/3! + x⁵/5! - x⁷/7! + ···
cos x = 1 - x²/2! + x⁴/4! - x⁶/6! + ···
ln(1+x) = x - x²/2 + x³/3 - x⁴/4 + ···
1/(1-x) = 1 + x + x² + x³ + ··· (|x|<1)
💡 记忆：eˣ全正、sin奇次、cos偶次、ln交替
📐 分部积分 & 换元
∫u dv = uv - ∫v du
口诀："反对幂三指" 排前留u（求导简化）
∫ f'(x)dx = f(x) + C （基本积分）
d/dx [∫ₐˣ f(t)dt] = f(x) （变上限求导）
💡 换元：三角代换 / 倒代换 / 指数代换
📊 微分方程通解
可分离：dy/y = k dx → y = Ceᵏˣ
一阶线性：y' + P(x)y = Q(x)
通解：y = e^{-∫Pdx}(∫Q·e^{∫Pdx}dx + C)
二阶常系数：r² + pr + q = 0
💡 r₁≠r₂ → y=C₁e^{r₁x}+C₂e^{r₂x}
📏 多元微分与极值
全微分：dz = ∂f/∂x dx + ∂f/∂y dy
链式法则：∂z/∂x = ∂f/∂u·∂u/∂x + ∂f/∂v·∂v/∂x
条件极值：拉格朗日乘数法
L = f(x,y) + λg(x,y)
💡 联立∂L/∂x=∂L/∂y=∂L/∂λ=0

🔺 一元函数微分学

  • 导数定义：f'(x₀)=lim[f(x₀+Δx)-f(x₀)]/Δx
  • 可导↔可微：Δy = AΔx + o(Δx)
  • 罗尔定理：f(a)=f(b) → ∃ξ∈(a,b), f'(ξ)=0
  • 拉格朗日：f(b)-f(a)=f'(ξ)(b-a)
  • 泰勒公式：f(x)=Σf⁽ⁿ⁾(x₀)/n!·(x-x₀)ⁿ + Rₙ

🔄 一元函数积分学

  • 牛顿-莱布尼茨：∫ₐᵇf(x)dx = F(b)-F(a)
  • 奇偶对称：偶函数面积加倍，积分限对称
  • 周期函数：∫₀ᴛf(x)dx 与起点无关
  • 旋转体体积：V=π∫ₐᵇf²(x)dx
  • 弧长公式：s=∫√(1+f'²)dx

🔷 多元函数积分学

  • 二重积分：∬f(x,y)dσ = ∫dx∫f(x,y)dy
  • 交换积分序：先y后x ↔ 先x后y
  • 极坐标变换：dσ = r dr dθ
  • 格林公式：∮Pdx+Qdy = ∬(∂Q/∂x-∂P/∂y)dσ
  • 高斯公式：∭(∂P/∂x+∂Q/∂y+∂R/∂z)dV

📈 无穷级数

  • 正项级数：比较法 / 比值法 / 根值法
  • 交错级数：莱布尼茨判别法（单调↓→0）
  • 条件收敛：∑|aₙ|发散，但∑aₙ收敛
  • 幂级数：收敛半径 R = lim|aₙ/aₙ₊₁|
  • 傅里叶级数：周期函数展开为三角级数

📊 数学建模

三大任务选型指南 — 准确识别题目类型，精准套用模型

🏷️

分类 / 评价类任务

题目要求：对方案打分、排序或分类
  • 层次分析法 AHP：主观权重，需构造判断矩阵，求特征值
  • TOPSIS 法：到理想解/负理想解距离，贴近度排序
  • 熵权法：客观赋权，信息熵越小权重越大
  • 模糊综合评价：模糊矩阵合成，适合边界不清的分类
  • 灰色关联分析：计算序列间关联度
📈

预测类任务

题目要求：根据历史数据预测未来走向
  • 灰色预测 GM(1,1)：小样本(~10个)，短期预测，精度检验用残差检验
  • 多元线性回归：大样本，多因素，检验 R²/F/t
  • 时间序列 ARIMA：季节性数据，差分使非平稳序列平稳
  • BP 神经网络：非线性映射，适合复杂规律
  • 插值与拟合：插值过所有点，拟合最优近似
⚙️

优化 / 决策类任务

题目要求：在限制条件下求最大值或最小值
  • 线性规划 LP：目标函数和约束均为线性，用单纯形法
  • 整数规划 ILP：变量取整（0-1规划/指派问题）
  • 非线性规划：罚函数法/梯度下降法求解
  • 多目标优化：理想点法/加权求和法/帕累托最优
  • 图论：最短路/最小生成树/网络流/TSP

🔍 模型选择速查

特征 推荐模型 关键参数 / 注意事项
样本量 < 20 灰色预测 GM(1,1) 级比检验，后验差比 C < 0.35
样本量 > 100，多因素 多元回归 / 机器学习 R² > 0.7，F 检验显著，共线性 VIF < 10
强周期性数据 时间序列 SARIMA 差分阶数 d，季节周期 s
变量为 0/1 决策 整数规划 / 遗传算法 用 Lingo 或 MATLAB 求解

☕ Java 核心技术

OOP · 集合框架 · 异常 · IO · 多线程 — 核心知识点全梳理

🧠 多态执行逻辑

编译看左边 / 运行看右边
父类引用指向子类对象时，只能调用父类声明的方法，实际执行子类重写后的方法
Animal a = new Dog(); a.eat(); // ✅ 编译看Animal，运行看Dog a.bark(); // ❌ 编译报错！Animal没有bark()
instanceof — 安全向下转型
先判断再强转，避免 ClassCastException
if (a instanceof Dog) { Dog d = (Dog) a; // 安全强转 d.bark(); }

📦 集合框架核心对比

List vs Set vs Map
List：有序可重复，有索引 → ArrayList(查快) / LinkedList(增删快)
Set：无序唯一 → HashSet(哈希去重) / TreeSet(自动排序)
Map：键值对，键唯一 → HashMap / TreeMap / LinkedHashMap
fail-fast 机制
ArrayList/HashMap 在迭代时若检测到结构变化，抛出 ConcurrentModificationException

⚠️ 异常处理机制

Checked vs Unchecked
Checked（编译时异常）：IOException, SQLException → 必须 try-catch 或 throws
Unchecked（运行时异常）：NullPointer, ArrayIndex, ClassCast → 可处理可不处理
异常执行顺序
try { 可能异常 } catch { 处理异常 } finally { 必定执行（关闭资源） }

🔌 IO 流体系

字节流 vs 字符流
字节流：InputStream/OutputStream → 二进制文件(图片/音频)
字符流：Reader/Writer → 文本文件(UTF-8)
缓冲流 / 转换流
BufferedReader/BufferedWriter → 高效读写；InputStreamReader → 字节→字符转换

🧵 多线程核心知识

线程创建方式
① 继承 Thread 类 → 重写 run()

🗄️ 数据库 SQL 核心语句

📋 DDL — 数据定义
CREATE TABLE t (id INT PRIMARY KEY, name VARCHAR(50));
ALTER TABLE t ADD age INT;
DROP TABLE IF EXISTS t;
CREATE INDEX idx_name ON t(name);
💡 DDL：自动提交，无法回滚
📝 DML — 数据操作
INSERT INTO t VALUES(1,'Alice');
UPDATE t SET name='Bob' WHERE id=1;
DELETE FROM t WHERE id=1;
SELECT * FROM t WHERE age > 18 ORDER BY age DESC;
💡 DELETE 可回滚，TRUNCATE 不可回滚
🔗 连接查询
INNER JOIN：两表匹配行
LEFT JOIN：左表全部+右表匹配
RIGHT JOIN：右表全部+左表匹配
子查询：SELECT * FROM t WHERE age > (SELECT AVG(age) FROM t);
💡 GROUP BY + HAVING 做分组聚合筛选
⚡ 事务与锁

📜 中国近代史纲要

救亡图存时间轴 — 各阶级探索历程与历史意义

1840
鸦片战争 — 近代史起点
中国开始沦为半殖民地半封建社会，《南京条约》割让香港岛、赔款2100万银元
1851–1864
农民阶级的探索
🌾 农民阶级
太平天国运动
洪秀全、杨秀清领导 · 颁布《天朝田亩制度》（废除土地私有）
《资政新篇》提出学习西方、发展资本主义
❌ 失败原因：农民阶级局限性，无法提出科学革命纲领
1860s–1890s
地主阶级的探索
🏛️ 地主阶级
洋务运动（自强·求富）
代表：曾国藩、李鸿章、左宗棠、张之洞
创办军事工业（江南制造总局）、民用工业、派遣留学生
❌ 失败标志：1895甲午战争北洋水师全军覆没
→ 证明"中学为体、西学为用"无法救国
1898
资产阶级维新派的探索
✊ 资产阶级维新派
戊戌变法（百日维新）
康有为、梁启超、谭嗣同 · 光绪帝支持
颁布《应诏统筹全局折》，提出政治、经济、文化改革
❌ 失败原因：改良道路在半殖半封中国行不通
→ 慈禧太后发动戊戌政变，谭嗣同就义
1911
资产阶级革命派的探索
🔥 资产阶级革命派
辛亥革命
孙中山 · 1894年成立兴中会 · 1905年同盟会（机关报《民报》）
纲领：三民主义（民族·民权·民生）
⚠️ 历史意义：推翻封建帝制，结束2000多年君主专制
❌ 局限性：未改变半殖半封社会性质，果实被袁世凯窃取
1919.5.4
五四运动 — 新民主主义革命开端
导火索：巴黎和会拒绝中国合理要求
意义：工人阶级登上历史舞台，马克思主义广泛传播
1921.7
中共一大 — 中国共产党成立
上海召开 · 毛泽东、董必武等13人 · 确定党的名称、纲领
意义：中国革命面貌焕然一新
1924–1927
第一次国共合作
国民革命 / 大革命时期
1924年孙中山改组国民党，实现第一次国共合作
1926年北伐战争，目标是推翻北洋军阀统治
⚠️ 1927年蒋介石"四一二"、汪精卫"七一五"反革命政变 → 合作破裂
1927–1937
土地革命战争时期
开辟农村包围城市的革命道路
1927年"八七会议"：确立土地革命和武装反抗方针
1931年中华苏维埃共和国成立
📍 1935年遵义会议：生死攸关转折点
确立毛泽东在党中央的领导地位，挽救了党、挽救了红军
1931/1937–1945
抗日战争时期
全民族抗战
1931年"九一八"事变 → 东北沦陷
1937年"七七"事变 → 全面抗战开始
1937年洛川会议：全面抗战路线，发动群众
抗日民族统一战线形成
中共是中流砥柱 · 毛泽东发表《论持久战》
1945年 抗战胜利
台湾回归祖国 · 中国国际地位提升
1945–1949
解放战争时期
重庆谈判（1945）→ 政治协商会议
1946年国民党进攻中原解放区 → 内战全面爆发
1947年刘邓大军挺进大别山 → 战略进攻转折
三大战役（辽沈·淮海·平津）：基本消灭国民党主力
推翻三座大山（帝国主义·封建主义·官僚资本主义）
1949.10.1
🇨🇳 中华人民共和国成立
毛泽东在天安门城楼宣布新中国成立
意义：中国人民从此站起来了！新民主主义革命取得基本胜利
